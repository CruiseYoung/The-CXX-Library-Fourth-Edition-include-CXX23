范围库是在C++20中添加的，但在C++23中有强大的扩展。范围库具有惰性算法，直接在容器上操作，并且可以组合。此外，大多数经典STL算法都具有范围挂坠，支持投影并提供额外的安全保证。

\filename{范围的组成}

\begin{cpp}
// rangesFilterTransform.cpp
...
#include <ranges>

std::vector<int> numbers = {1, 2, 3, 4, 5, 6};

auto results = numbers | std::views::filter([](int n){ return n % 2 == 0; })
				       | std::views::transform([](int n){ return n * 2; });
				       
for (auto v: results) std::cout << v << " "; // 4 8 12
\end{cpp}

必须从左到右读这个表达式。管道符号|代表函数组合:首先，所有偶数都可以通过(std::views::filter([](int n)\{return n \% 2 == 0;\}))。之后，每个剩余的数字映射到双倍结果(std::views::transform([](int n)\{return n * 2;\}))。
