\href{https://en.cppreference.com/w/cpp/ranges/iota_view}{std::views::iota} is a range factory for creating a sequence of elements by successively incrementing an initial value. This sequence can be finite or infinite. Thanks to the function, I can find the first 20 prime numbers starting with 1000000.

\filename{Finding 20 prime numbers starting with 1000000}

\begin{cpp}
// rangesLazy.cpp
...
#include <ranges>

bool isPrime(int i) {
	for (int j=2; j*j <= i; ++j){
		if (i % j == 0) return false;
	}
	return true;
}
											// (1)
std::cout << "Numbers from 1000000 to 1001000 (displayed each 100th): " << "\n";
for (int i: std::views::iota(1000000, 1001000)) {
	if (i % 100 == 0) std::cout << i << " ";
}
											// (2)
auto odd = [](int i){ return i % 2 == 1; };
std::cout << "Odd numbers from 1000000 to 1001000 (displayed each 100th): " << "\n";
for (int i: std::views::iota(1000000, 1001000) | std::views::filter(odd)) {
	if (i % 100 == 1) std::cout << i << " ";
}
	
											// (3)
std::cout << "Prime numbers from 1000000 to 1001000: " << '\n';
for (int i: std::views::iota(1000000, 1001000) | std::views::filter(odd)
											   | std::views::filter(isPrime)) {
	std::cout << i << " ";
}
											// (4)
std::cout << "20 prime numbers starting with 1000000: " << '\n';
for (int i: std::views::iota(1000000) | std::views::filter(odd)
									  | std::views::filter(isPrime)
									  | std::views::take(20)) {
	std::cout << i << " ";
}
\end{cpp}

Here is my iterative strategy:

\begin{enumerate}
\item

\item 
A prime is an odd number; therefore, I remove the even numbers.

\item 
The predicate isPrime returns if a number is a prime. I get 75 primes, but I only want to have 20.

\item 
I use std::iota as an infinite number factory, starting with 1000000, and ask precisely for 20 primes.
\end{enumerate}
