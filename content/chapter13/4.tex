字符串支持众所周知的比较操作符==、!=、<、>、>=。两个字符串的比较发生在它们的元素上。

\filename{字符串的比较}

\begin{cpp}
// stringComparisonAndConcatenation.cpp
...
#include <string>
...

std::string first{"aaa"};
std::string second{"aaaa"};

std::cout << (first < first) << '\n'; // false
std::cout << (first <= first) << '\n'; // true
std::cout << (first < second) << '\n'; // true
\end{cpp}
